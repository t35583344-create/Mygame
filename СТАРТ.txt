⭐ ФАЗА 1: ТЕСТИРОВАНИЕ ЛОКАЛЬНО (опционально)
────────────────────────────────────────────

Если хотите сначала протестировать локально на своем компе:

1. Откройте папку Downloads в PowerShell/CMD
2. Выполните:
   npm install
   node server.js

3. Откройте index.html в браузере
4. Введите:
   - Имя: ваше имя
   - Server: localhost:8081
5. Нажмите "Join Game"

⭐ ФАЗА 2: РАЗВЕРТЫВАНИЕ ОНЛАЙН (для игры с другом)
──────────────────────────────────────────────

ВЫБЕРИТЕ ОДИН СПОСОБ:

🌐 СПОСОБ 1: Replit (САМЫЙ ПРОСТОЙ, рекомендуется)
─────────────────────────────────────────────────

1. Откройте https://replit.com
2. Создайте аккаунт (можно через Google)
3. Нажмите "+ Create Repl"
4. Выберите Node.js
5. Загрузите эти файлы:
   • server.js
   • index.html
   • package.json
6. Нажмите "Run" вверху
7. Ждите, пока запустится (30 секунд)
8. Вы увидите URL типа:
   👉 https://трейлер-block-builder.replit.dev:3000

СКОПИРУЙТЕ ЭТО БЕЗ https://
👉 трейлер-block-builder.replit.dev:3000

9. Откройте index.html локально
10. Введите:
    - Имя: ваше имя
    - Server: (скопированный URL без https://)
11. Нажмите "Join Game"

✅ ВЫ ПОДКЛЮЧЕНЫ!

12. ОТПРАВЬТЕ ДРУГУ URL (без https://):
    👉 трейлер-block-builder.replit.dev:3000

13. Друг откроет index.html и введет:
    - Имя: его имя
    - Server: (ВАШ URL)
    - Чекбокс "Use Secure Connection" оставить включенным
14. Нажмет "Join Game"

✅ МОЖНО ИГРАТЬ!

─────────────────────────────────────────────

🚂 СПОСОБ 2: Railway (более надежный)
──────────────────────────

1. Откройте https://railway.app
2. Создайте аккаунт
3. Создайте новый проект
4. Выберите "Deploy from GitHub" или загрузите файлы
5. Railway автоматически запустит сервер
6. Получите публичный URL
7. Используйте его вместо localhost

─────────────────────────────────────────────

📘 СПОСОБ 3: Heroku
──────────────────

1. Откройте https://www.heroku.com
2. Создайте аккаунт
3. Создайте новое приложение
4. Свяжите с GitHub или используйте Heroku Git
5. Разверните приложение
6. Используйте предоставленный URL

✅ СПИСОК КОНТРОЛЯ:
──────────────────

Перед игрой проверьте:

□ Сервер запущен на Replit/Railway/Heroku
□ Адрес сервера скопирован правильно (без https://)
□ index.html открыт в браузере
□ Введено ваше имя
□ Введен адрес сервера в поле "Server address"
□ Галочка "Use Secure Connection" включена (для онлайн)
□ Нажата кнопка "Join Game"

🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:
───────────────────────────

❌ "Connection Error"?
→ Откройте check_connection.html
→ Введите адрес сервера
→ Нажмите "Проверить соединение"
→ Это покажет, в чем проблема

❌ Не видит друга?
→ Обождите 1-2 секунды, обновите страницу (F5)
→ Убедитесь, что друг ввел ВАШУ ссылку, а не свою

❌ Блоки не синхронизируются?
→ Проверьте интернет
→ Перезагрузите обоим index.html (F5)

❌ Сервер Replit не запускается?
→ Проверьте, скопировали ли все файлы
→ Убедитесь, что package.json вводится правильно
→ Нажмите "Run" еще раз
